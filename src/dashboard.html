配置指南

1. 获取免费 API 密钥

使用本中转站需要上游提供商的 API 密钥，以下均为免费注册：

  • OpenRouter - openrouter.ai/keys - 25+ 免费模型，推荐首选
  • Groq - console.groq.com - 超快推理，14400请求/天
  • Cerebras - cloud.cerebras.ai - 超快推理，14400请求/天
  • SambaNova - cloud.sambanova.ai - $5免费额度
  • Google AI Studio - aistudio.google.com - Gemini 免费
  • Mistral - console.mistral.ai - 10亿token/月免费
  • GitHub Models - github.com/marketplace/models - GPT-4o 免费
  • NVIDIA NIM - build.nvidia.com - 40请求/分钟
  • Cohere - cohere.com - Command 模型免费

2. 配置提供商密钥

编辑 .env 文件或使用管理面板管理提供商：

# 格式：名称|基础URL|API密钥|权重|启用
PROVIDERS=openrouter|https://openrouter.ai/api/v1|your-key|5|true

3. 使用 API

使用 OpenAI 兼容格式发送请求：

curl http://localhost:3000/v1/chat/completions \
  -H "Authorization: Bearer sk-proxy-demo-key-123" \
  -H "Content-Type: application/json" \
  -d '{"model":"deepseek/deepseek-v4-flash:free","messages":[{"role":"user","content":"Hello!"}]}'
